Sort, name, describe and draw 2D shapes

What are 2D shapes?
2D shapes are flat shapes that you can draw or see on paper. They only have length and width, no height or depth.
Common 2D shapes:
Square:
  • A square has 4 sides that are all the same length and 4 corners that are sharp.
  • Example: A checkerboard or a window is shaped like a square.
Common 2D shapes:
Rectangle:
  • A rectangle has 4 sides. The opposite sides are the same length, but the sides are not all equal.
  • Example: A door or a book is shaped like a rectangle.
Common 2D shapes:
Circle:
  • A circle is round with no sides or corners.
  • Example: A clock or a pizza is shaped like a circle.
Common 2D shapes:
Pentagon (regular):
  • A regular pentagon has 5 sides that are all the same length.
  • Example: A house roof can look like a pentagon.
Common 2D shapes:
Pentagon (irregular):
  • An irregular pentagon also has 5 sides, but the sides are not the same length.
  • Example: It might look like a weirdly shaped sign or drawing.
Common 2D shapes:
Hexagon (regular):
  • A regular hexagon has 6 sides, and all the sides are the same length.
  • Example: A honeycomb is made of hexagons
Common 2D shapes:
Hexagon (irregular):
  • An irregular hexagon also has 6 sides, but the sides are not all the same length.
  • Example: Some puzzle pieces can be shaped like an irregular hexagon.
