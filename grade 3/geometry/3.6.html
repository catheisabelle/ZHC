Finding Positions on Grids

Let’s learn about finding positions on grids!
A grid is a set of squares arranged in rows and columns, like a checkerboard. We use numbers and letters to describe positions on the grid.
Rows and Columns
Rows go across (side to side), and columns go up and down.

Example: In a grid, the rows might have numbers (1, 2, 3…) and the columns might have letters (A, B, C…).

Finding a Square Using Coordinates
To find a square, look at the row and column it’s in.

Example: If the position is B3, start in column B and go to row 3. That’s your square!
