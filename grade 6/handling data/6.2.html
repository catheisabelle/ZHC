Finding the Mode, Range, Median, and Mean of a Data Set

Mode of a Data Set
Mode:
The mode is the number that appears most frequently in a data set.

Example of Mode:
  • Data: 3, 5, 7, 5, 2, 5, 8
  • The mode is 5 because it appears three times, more than any other number.
Range of a Data Set
Range:
The range is the difference between the highest and lowest values in the data set.
Formula: Range = Highest value - Lowest value

Example of Range:
  • Data: 8, 3, 12, 5, 10
  • Highest value = 12, Lowest value = 3
  • Range = 12 - 3 = 9
Median of a Data Set
Median:
The median is the middle number of a data set when the numbers are arranged in ascending (smallest to largest) or descending (largest to smallest) order.

Steps to find the Median:
  • Sort the numbers from least to greatest.
  • If there is an odd number of numbers, the median is the middle number.
  • If there is an even number of numbers, the median is the average of the two middle numbers.
Example (Odd Number of Data):
  • Data: 5, 3, 8, 6, 7
  • Sort: 3, 5, 6, 7, 8
  • The median is 6 (the middle number).
Example (Even Number of Data):
  • Data: 3, 8, 7, 5, 6, 9
  • Sort: 3, 5, 6, 7, 8, 9
