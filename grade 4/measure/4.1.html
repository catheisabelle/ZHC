Using Standard Metric Units and Decimal Notation

What are Standard Metric Units?
Standard metric units are a way of measuring things like length, weight, and capacity using numbers that everyone can understand.

Examples:
  • Length: meter (m), centimeter (cm), millimeter (mm), kilometer (km)
  • Weight: kilogram (kg), gram (g)
  • Capacity: liter (l), milliliter (ml)
What is Decimal Notation?
Decimal notation is a way to write numbers with a decimal point, like 1.5 or 2.3. It helps us measure things more precisely.

Example: If something is 1.3 meters long, the 1 is the full meter, and the .3 shows part of a meter (about 1/3 of a meter).

How to Use Metric Units: Length
Measuring length: To measure how long something is, you might use meters for big things like a table, or centimeters for smaller things like a pencil.

How to Use Metric Units: Weight
Measuring weight: If you weigh a book, you might use kilograms. For lighter things, you use grams.

How to Use Metric Units: Capacity
Measuring capacity: To measure how much liquid is in a bottle, you would use liters. For smaller amounts, like a drink, you use milliliters.
